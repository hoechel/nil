static method Copy_Files [line 124]

	

    

    

		static void Copy_Files(

$args, [bool
$file_protection = TRUE], [bool
$erase_comments = FALSE])
    

    


	
		Overwrites target file contents with source file contents

By validating the passed pairs of source and target filenames,  by checking the existance of source files, and by checking the  existance of target files AND the setting of file protection flag,  this method will ensure the successful completion of the copy  process and will not overwrite the target file content, if it is  protected. Otherwise the runtime will be stopped by throwing an  exception. It will also erase comments of the source file content  before the overwriting process starts, if the erase comments flag  demands it.
